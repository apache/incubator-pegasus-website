HTTP 接口

介绍

Pegasus 为 MetaServer，ReplicaServer 提供了 HTTP 接口，用于查看集群信息，修改配置等。

由于集群元数据由主 MetaServer 维护，当访问备 MetaServer 的元数据查询接口时，会自动重定向至主 MetaServer 对应的接口。 举个例子，假设 127.0.0.1:34601 和 127.0.0.1:34602 分别是主备 MetaServer，当访问 127.0.0.1:34602/meta/cluster时， 会自动跳转（通过 HTTP code 307）到 127.0.0.1:34601/meta/cluster。

接口

所有接口的结果都以 JSON 格式返回。使用浏览器查看结果时，建议使用 Chrome 插件 JSON Viewer 提升阅读体验。
